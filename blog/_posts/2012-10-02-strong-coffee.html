As of late, there's a lot of great stuff I like about what Microsoft has been up to. They're more open, more transparent, and more connected than they've ever been - and I think the result of all this is a more relevant company than they've been in a long time. Today, Microsoft announced TypeScript as one more attempt to reach out to developers, make their lives easier, and make the web a more open place and a platform that's easier than ever to develop for.

Miguel de Icaza does a great job rounding up the pros and cons of this release, so I won't rehash it here. I think some of the cons and barriers to adoptability that he mentions Linux and Mac developers will face shouldn't be barriers for very long. Some of the initial syntax definitions available already make this more attractive to non Windows developers.

All in all - I think it's a worthy contribution to the landscape and, along with some other excellent tools addressing similar problems and having similar goals, it only adds to developer's abilities to efficiently write solid code.
